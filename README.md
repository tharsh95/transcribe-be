Follow the steps below to set up and run the project locally.

## Steps

1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd <repository-folder>
   ```
2. **Install Dependencies**
    ```bash
    npm install
    ```
3. **Start the Server**
    ```bash
    npm run start:dev
    ```
## Backend is running on localhost:3000
